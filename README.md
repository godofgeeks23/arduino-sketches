# Arduino Programming


## Useful functions

```
setup() # Setup function
loop()  # Main
pinMode(<int>, <int>) # Pin: 13, Mode: OUTPUT
digitalWrite(<int>, <int>) # Pin: 13, Value: HIGH
delay(<int>) # Time: 1000ms
Serial.begin(<int>) # Baud rate: 9600
Serial.available()
Serial.read()
Serial.println()

```
